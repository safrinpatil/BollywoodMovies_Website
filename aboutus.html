About Us

Safrin Patil

Hello! My name is Safrin Patil, and I am an MS-BAIS student at the University of South Florida, currently enrolled in ISM 6225 - Distributed Information Systems. I have a strong passion for web development and user interface design. This website is a part of my coursework, and it showcases my ability to create a data-driven, interactive web application using HTML, CSS, and JavaScript.

Experience Developing This Website

Developing this website has been an enriching experience. It allowed me to apply my knowledge of web technologies and improve my skills in creating a user-friendly interface. The project involved several key steps, including:

  • Planning the website layout and structure based on the given assignment requirements.
  • Researching and gathering relevant data to populate the website content.
  • Designing and implementing the HTML and CSS to create a visually appealing and responsive design.
  • Adding interactivity using JavaScript, including creating dummy CRUD operations and data visualizations.
  • Ensuring that the website is easy to navigate and use, with a consistent look and feel across all pages.

Technologies Used

  • HTML5: For structuring the content and layout of the website.
  • CSS3: For styling the website and ensuring it is visually appealing and responsive.
  • JavaScript: For adding interactivity and dynamic content, including dummy CRUD operations.
  • Bootstrap: For creating a responsive design and utilizing pre-built components.
  • Chart.js: For creating interactive data visualizations on the Data Analytics page.
  • GitHub: For version control and hosting the source code of the website.
